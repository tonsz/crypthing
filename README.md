# How to Run 

- Open the directory on VSCode 
- Make sure to have Python installed
- Make sure to install the dependencies 
    - tkinter
    - pyperclip 
    - cryptography
  Install the dependencies by typing on the terminal 
  ` pip install tkinter`
  ` pip install pyperclip`
  ` pip install cryptography `
- Type on terminal
 ` python main.py `
