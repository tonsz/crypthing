# How to Run 

- Open the directory on VSCode 
- Make sure to have Python installed
- Make sure to install the dependencies 
    - tkinter
    - pyperclip 
    - cryptography
  Install the dependencies by typing on the terminal 
  ```vscode 
    pip install tkinter
    ```
  ```vscode 
    pip install pyperclip
    ```
  ```vscode 
    pip install cryptography
    ```
- Type on terminal
  ```vscode 
    python main.py
    ```
